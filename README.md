# Tic Tac Toe Game 🎮

## 📌 Project Description

This is a simple Tic Tac Toe game developed using JavaScript.
The game allows two players to play turn-by-turn on a 3x3 grid and checks for the winner after each move.

## 🚀 Features

* Two player game
* Interactive game board
* Winner detection
* Draw detection
* Reset game option

## 🛠️ Technologies Used

* HTML
* CSS
* JavaScript

## ▶️ How to Run the Project

1. Clone the repository
2. Open the project folder
3. Run the `index.html` file in your browser
